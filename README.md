# PWA版本QQ音乐

  > 这是一个采用PWA与百度开发的PWA框架Lava制作的webApp
  
  > 该APP引入了页面骨架屏与Service Worker离线缓存,让webApp接近了原生的体验
  
  > 当用户访问该网站第二次的时候还允许将站点添加至主屏幕，这个是 PWA 提供的一项重要功能。
  
  > 由于商业利益与职业道德关系已屏蔽部分接口，改项目仅供学习使用
